# 🌐 Network Scanner Dashboard

## 📌 Overview
The Network Scanner Dashboard is a Python and Flask-based web application that scans a target IP address using Nmap and displays open ports in a user-friendly dashboard.

## 🚀 Features
- Scan any IP address
- Detect open ports
- Display scan results in a web dashboard
- Download scan report
- Simple and responsive interface

## 🛠 Technologies Used
- Python
- Flask
- Nmap
- python-nmap
- HTML
- CSS

## ▶️ How to Run
1. Install Python
2. Install Nmap
3. Install dependencies
```bash
pip install flask python-nmap
```
4. Run the application
```bash
python app.py
```
5. Open your browser and visit:
```
http://127.0.0.1:5000
```
